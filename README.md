# Writer Helper 文字工作者助手

提供文字工作者常用的功能。

- 网络视频音频下载功能：小红书、哔哩哔哩、优酷、爱奇艺、网易云
- ruiwen.com 文章下载功能
- 学生成绩管理功能
- 豆包 AI 跳转功能

## 支持平台

- MacOS
- Windows10 及以上

## 实现方式

- golang wails 实现跨平台
- lux, ffmpeg 实现网络音视频下载

## 开发

`make run`

## 编译

- windows 10 及以上: `make build_windows`
- MacOS: `make build`
